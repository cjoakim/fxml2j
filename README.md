#### Setup

Add fxml2j to your project or package.json file:
```
npm install fxml2j
```

Require fxml2j in your code:
```
Fxml2j  = require("fxml2j").Fxml2j
```

Note: this library is implemented with CoffeeScript, and these examples are also in CoffeeScript.

#### Fxml2j


``` 
fx = new Fxml2j()

```
